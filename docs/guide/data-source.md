# 数据源说明

本文档介绍 MoeKoe Music 中可切换的两种数据源：正式版与概念版的区别和使用注意事项。

## 数据源类型
对应kugou手机版中的对应版本

## 切换方式

1. 打开应用内"设置"页面
2. 找到"数据源"选项
3. 选择"正式版"或"概念版"
4. 保存设置并重启应用

### 使用正式版时

::: warning 重要提示
正式版仅供测试和体验功能使用，可能会遇到以下问题：
:::

- ⚠️ **功能异常**：某些功能可能无法正常工作
- ⚠️ **数据不准确**：歌曲信息、专辑封面等可能出现错误
- ⚠️ **播放问题**：可能出现无法播放、卡顿等情况
- ⚠️ **接口变更**：接口可能随时调整，导致功能失效
- ⚠️ **无法保证修复**：正式版问题可能不会及时修复

### 建议

- **普通用户**：强烈建议使用正式版数据源
- **遇到问题时**：请先切换回概念版，确认是否为数据源导致
- **反馈问题时**：请注明使用的数据源版本

## 常见问题

### 如何判断当前使用的是哪个数据源？

在设置页面的"数据源"或"关于"部分可以查看当前使用的数据源版本。

### 切换数据源会影响数据吗？

不会。切换数据源只影响在线音乐的获取,不影响用户个人数据。

**建议：遇到问题请切换回概念版使用。**
