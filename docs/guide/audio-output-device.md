# 音频输出设备

该功能用于指定 MoeKoe Music 的播放输出设备，让应用的声音固定从你选择的设备输出，或“跟随系统默认”自动切换。

::: warning 系统与权限限制
该功能受系统音频架构与驱动实现影响；
在获取设备信息（设备列表/名称）或切换输出设备时，系统可能会弹出权限申请。
若系统权限/策略限制导致应用无法获取设备信息或切换输出，可能出现“设备列表为空/名称不显示”或“选择后不生效”等情况。
:::

## 使用方式

在设置页找到“音频输出设备”，通常会看到两类选项：

- 跟随系统默认：应用始终使用当前系统默认输出设备
- 指定设备：应用优先使用你选择的输出设备进行播放

## 常见用法

- 会议/录屏时固定输出到虚拟声卡或指定耳机，避免被系统默认切换影响
- 多设备环境下固定输出到桌面音箱或显示器音箱

## 设备不可用时会怎样

当你指定的输出设备被拔出、断开或系统不再可用时，应用的表现取决于平台与设置，常见情况包括：

- 暂停播放并提示你重新选择设备
- 自动回退到系统默认输出设备继续播放

如果你更在意“避免意外外放”，建议同时开启“输出设备变化自动暂停”，详见 [/guide/auto-pause-on-output-device-change](/guide/auto-pause-on-output-device-change)。

## 设备列表为空/名称不显示

在部分系统（尤其是 macOS）上，为了获取音频设备列表或显示设备名称，系统可能会弹出麦克风权限请求。这通常用于读取设备信息而非录音。

- 相关问题可参考 [/faq/microphone-permission](/faq/microphone-permission)
