# 音频响度平衡说明

本文档介绍 MoeKoe Music 中的音频响度平衡功能，帮助您获得更一致的音量体验。

## 功能简介

音频响度平衡（Audio Normalization）是一种自动调节音频音量的技术，可以让不同歌曲在播放时保持相似的响度水平，避免频繁调节音量。

## 为什么需要响度平衡？

在日常听歌过程中，您可能会遇到以下问题：

- 🔊 **音量不一致**：不同歌曲、专辑的音量差异很大
- 📀 **制作年代差异**：老歌和新歌的母带响度标准不同
- 🎵 **曲风差异**：古典音乐和流行音乐的动态范围不同
- ⚡ **切歌惊吓**：切换歌曲时音量突然变大或变小

响度平衡功能可以智能分析每首歌的响度，自动调整到一致的音量水平。

## 功能特点

### 智能分析
- 自动检测歌曲的平均响度和峰值
- 根据国际标准（如 LUFS）计算目标响度
- 保持音乐的动态范围，不损失音质

### 无损调节
- 仅调整播放音量，不修改源文件
- 实时处理，无需预先转码
- 保留音频的原始品质

## 使用方法

### 开启响度平衡

1. 打开应用内"设置"页面
2. 找到"声音"设置
3. 开启"响度平衡"或"音量标准化"开关
4. 设置生效，无需重启

### 调整目标响度

某些版本支持自定义目标响度级别：

- **标准模式**：适合大多数用户，平衡音量适中
- **响亮模式**：整体音量偏大，适合嘈杂环境
- **安静模式**：整体音量偏小，适合夜间或办公环境

## 使用场景

### 适合使用响度平衡的场景

✅ **随机播放/电台模式**：曲风和年代差异大的歌单

✅ **开车/跑步**：避免频繁调节音量

✅ **背景音乐**：需要稳定音量的场合

✅ **混合歌单**：包含不同来源、不同年代的歌曲

### 不建议使用响度平衡的场景

❌ **专辑完整收听**：专辑内部的音量设计是制作人的艺术意图

❌ **古典音乐**：需要保留大动态范围和强弱对比

❌ **现场录音**：现场的动态变化是音乐的一部分

❌ **发烧友精听**：希望完全还原制作人的原始意图

## 技术说明

### 响度标准

MoeKoe Music 的响度平衡功能参考以下国际标准：

- **LUFS** (Loudness Units relative to Full Scale)：国际标准响度单位
- **目标响度**：通常为 -14 LUFS（Spotify、Apple Music 等流媒体平台标准）
- **真峰值限制**：防止削波失真

### 处理流程

1. **分析阶段**：计算歌曲的综合响度（Integrated Loudness）
2. **增益计算**：计算达到目标响度所需的增益值
3. **实时应用**：在播放时实时应用增益调整
4. **峰值保护**：确保调整后不会产生削波失真

## 常见问题

### 开启响度平衡后音质会下降吗？

不会。响度平衡只是调整播放音量，使用的是数字增益控制，不会影响音频的采样率、比特率等参数，音质保持不变。

### 为什么开启后有些歌还是很响/很轻？

可能的原因：
- 歌曲文件本身的动态范围很大
- 源文件的响度信息异常
- 需要积累播放数据进行更准确的分析

建议切换到下一首歌再试，或调整目标响度设置。

### 响度平衡和音量增强有什么区别？

- **响度平衡**：让所有歌曲音量保持一致，有的歌增大、有的歌减小
- **音量增强**：统一增大所有歌曲音量，可能导致失真

响度平衡更科学、更安全，推荐使用。

### 会影响均衡器（EQ）设置吗？

不会。响度平衡和均衡器是独立的功能：
- 响度平衡调整整体音量
- 均衡器调整频率响应

两者可以同时使用，互不干扰。

## 最佳实践建议

1. **日常听歌**：建议开启响度平衡，获得更舒适的体验
2. **专辑精听**：建议关闭响度平衡，尊重原始制作意图
3. **混合场景**：根据播放列表的特点灵活切换
4. **音量设置**：开启响度平衡后，建议将系统音量设置在 60-80% 左右
