# 全屏歌词复制功能

本文档介绍 MoeKoe Music 的一个隐藏功能：在全屏歌词界面快速复制当前歌词内容。

## 功能简介

在全屏歌词界面，您可以使用快捷键一键复制当前显示的所有歌词内容，包括原文、翻译和音译（如果有）。这是一个隐藏的便捷功能，适合需要分享或保存歌词的场景。

## 使用方法

### 进入全屏歌词界面

1. 播放任意歌曲
2. 点击进入全屏歌词模式
3. 歌词将以全屏方式显示

### 复制歌词

在全屏歌词界面下，使用以下快捷键即可复制歌词：

- **Windows / Linux**：按下 `Ctrl + C`
- **macOS**：按下 `Cmd + C`

复制成功后，屏幕上会显示提示消息确认操作完成。

## 功能特点

### 完整内容复制

✅ **原文歌词**：包含歌曲的原始歌词文本

✅ **翻译歌词**：如果歌曲有翻译，将一并复制

✅ **音译歌词**：如果歌曲有音译（如罗马音），也会包含在内

### 格式良好

✅ **换行分隔**：每行歌词独立一行，便于阅读

✅ **结构清晰**：原文、翻译、音译按顺序排列

✅ **即用即粘贴**：复制后可直接粘贴到任何文本编辑器或聊天工具

### 跨平台支持

✅ **Windows**：支持标准 Ctrl+C 快捷键

✅ **Linux**：支持标准 Ctrl+C 快捷键

✅ **macOS**：支持标准 Cmd+C 快捷键

## 注意事项

### 功能限制

⚠️ **仅全屏模式**：此功能只在全屏歌词界面生效，普通播放界面不支持

⚠️ **需要歌词数据**：歌曲必须有歌词才能复制，纯音乐无法复制内容

⚠️ **完整复制**：功能复制所有歌词内容，不支持选择性复制单句

## 常见问题

### 为什么按快捷键没有反应？

可能的原因：
- 不在全屏歌词界面（请先进入全屏模式）
- 当前歌曲没有歌词
- 系统剪贴板权限被限制
- 快捷键与其他软件冲突

**解决方法**：
1. 确认已进入全屏歌词界面
2. 检查歌曲是否有歌词显示
3. 检查系统剪贴板权限设置

### 为什么有些歌曲复制的内容很少？

可能的原因：
- 歌曲只有原文歌词，没有翻译和音译
- 歌词数据来源不完整
- 部分歌词受版权保护，无法获取完整内容

这是正常现象，取决于歌曲的歌词数据完整度。

### 复制的歌词包含时间轴吗？

不包含。复制功能只提取纯文本歌词内容，不包含 LRC 格式的时间轴信息。如果需要带时间轴的歌词文件，请使用其他专业工具。

### 可以复制部分歌词吗？

目前不支持。快捷键会复制当前歌曲的所有歌词内容。如果只需要部分歌词，建议：
1. 先复制全部歌词
2. 粘贴到文本编辑器
3. 手动选择需要的部分

## 隐藏功能的发现

这个功能被设计为"隐藏功能"，没有在界面上显示提示或按钮，只能通过快捷键触发。这是为了保持全屏歌词界面的简洁美观，同时为熟悉快捷键的用户提供便利。
