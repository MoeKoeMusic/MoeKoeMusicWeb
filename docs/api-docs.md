# API 文档

MoeKoe Music 基于[MakcRe/KuGouMusicApi](https://github.com/MakcRe/KuGouMusicApi) 开发，使用了一套完整的官方 API 接口。如果您对 API 细节感兴趣，或者想要了解如何与 MoeKoe Music 进行交互，可以参考以下文档。
